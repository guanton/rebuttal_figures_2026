# Rebuttal Figures


## Included Files

* **`eb_interpolation_figure_pca.pdf`**
    WLM's performance for interpolating unseen marginals for the leave-one-out (LOO) task on the Embryoid Body (EB) dataset is visualized (in 2D PCA space) on each of the three unseen marginals at times t1, t2, and t3. 

* **`eb_interpolation_figure.pdf`**
    WLM's performance for interpolating unseen marginals for the leave-one-out (LOO) task on the Embryoid Body (EB) dataset is visualized (in UMAP space) on each of the three unseen marginals at times t1, t2, and t3. 

* **`oceans_interpolation_figure.pdf`**
    WLM's performance for interpolating four unseen marginals on the Gulf of Mexico ocean vortex dataset is visualized.
